## What is Increment?

**Increment** = Time added after each move

**Examples:**
- **15+10**: Start with 15 minutes, get +10 seconds per move
- **5+0**: Start with 5 minutes, no time added
- **10+5**: Start with 10 minutes, get +5 seconds per move

**Why use increment?**
- Prevents time scrambles
- Allows strategic thinking
- Balances speed and quality
