## How to Use

### Using Buttons
1. Click **"Undo Move"** button to take back the last move
2. Click **"Redo Move"** button to replay an undone move

### Using Keyboard Shortcuts ‚å®Ô∏è
- **Ctrl + Z**: Undo the last move
- **Ctrl + Y**: Redo an undone move
- **Ctrl + N**: Start a new game

### Using Menu
1. Click **Game** menu at the top
2. Select **Undo Move** or **Redo Move**

## Tips üí°

‚úÖ **You can undo up to 100 moves back**
‚úÖ **All special moves are supported** (castling, en passant, promotion)
‚úÖ **Making a new move clears redo history** (standard behavior)
‚úÖ **Starting a new game clears all history**

## Examples

### Undo a Mistake
1. Accidentally moved your queen into danger? 
2. Press **Ctrl+Z** immediately
3. Your queen is back safe!

### Explore Different Strategies
1. Try a move to see how it looks
2. Press **Ctrl+Z** if you don't like it
3. Try a different move instead

### Review Your Game
1. Finished a game? 
2. Press **Ctrl+Z** repeatedly to review moves
3. Press **Ctrl+Y** to replay forward

## Visual Feedback

- Move counter updates when undoing/redoing
- Move history list reflects current position
- Last move highlighting updates
- Turn indicator shows correct player

## Important Notes ‚ö†Ô∏è

- If you make a new move after undoing, you cannot redo past that point
